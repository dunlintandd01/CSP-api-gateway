# MCGame

## Tech Stack

- http framework using [Nest.js](https://docs.nestjs.com/)
- database ORM using [TypeORM](https://github.com/typeorm/typeorm)(Mysql)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
